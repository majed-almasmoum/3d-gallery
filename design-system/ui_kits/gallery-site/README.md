# Gallery Portfolio Site — UI Kit

## Overview
Interactive bilingual (AR/EN) portfolio site for Majed Almasmoum's 3D-printed art gallery.

## Screens
1. **Homepage** — Hero + featured works grid
2. **Gallery** — Full artwork grid with filter tags
3. **Artwork Detail** — Single work detail view
4. **Commission** — Inquiry form

## Components
- `Header.jsx` — Blur nav with lang toggle
- `GalleryGrid.jsx` — Masonry-style artwork grid
- `ArtworkDetail.jsx` — Detail panel / lightbox
- `CommissionForm.jsx` — Inquiry form
- `Footer.jsx` — Footer with links

## Design Width
1280px desktop
